# Extension Icons

Place the following icon files in this directory:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

You can create these icons using any image editor. The icons should represent the ChatGPTGraphs extension (e.g., a graph/chart icon).

For now, you can use placeholder images or create simple icons. The extension will work without icons, but Chrome will show a default icon.
